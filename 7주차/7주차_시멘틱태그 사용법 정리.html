시맨틱 태그 살펴보기

<header>

페이지나 섹션의 머리말을 표현하는 태그로 페이지 제목, 로고, 페이지를 소개하는 간단한 글 등을 포함.

<nav>

hyperlink들을 모아 보여주는 영역을 나타내는 태그로 페이지 내의 목차, 메뉴, 다른 페이지로의 링크 등을 포함.

<section>

문서의 장 또는 절을 구성하는 태그로 헤딩 태그를 주로 포함. 페이지 내에 여러 section을 둘 수 있으며, 더 적합한 의미를 가진 태그가 없을 때 사용.

<article>

독립적인 콘텐츠를 담은 영역을 나타내는 태그로 블로그의 포스트, 매거진이나 뉴스 기사 등을 포함. article에 담는 내용이 많은 경우 여러 section으로 나누어 담을 수 있음.

<aside>

웹 페이지 본문의 흐름에서 벗어나거나 간접적으로 연관된 내용을 나타내는 태그로 페이지의 오른쪽이나 왼쪽에 주로 배치.

<footer>

꼬리말 영역을 표시하는 태그로 작성자, 저작권 정보 등의 내용을 포함.

Semantic Block Tags

블록을 구성하는 HTML5 시맨틱 태그로 <figure>, <details> 태그 등이 있다. <figure> 태그는 본문에 삽입된 그림을 블록화하며, <details> 태그는 상세 정보를 담아 감추거나 보이게 할 수 있다.
